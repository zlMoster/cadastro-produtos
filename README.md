# Cadastro de Produtos

Nome da dupla: Gustavo Henrique Nunes de Oliveira / Matheus Alves Costa

Projeto de cadastro de produtos simples usando React e Vite.

## O que tem no projeto

- Cadastro de produtos com nome, preço e categoria
- Criação de novas categorias durante o cadastro
- Exibição de produtos cadastrados em uma lista
- Backend local simulando API com `json-server`

## Tecnologias usadas

- React
- Vite
- json-server
- @vitejs/plugin-react


## Estrutura principal

- `src/App.jsx` - componente principal
- `src/components/ProductForm.jsx` - formulário de cadastro
- `src/components/ProductList.jsx` - lista de produtos
- `db.json` - dados usados pelo `json-server`
